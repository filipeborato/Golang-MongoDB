# Golang-MongoDB

### Instalation and Testing

Runing project:

```
go run main.go
```

Testing Route:

```
http://localhost:8080/populate-test
```

Get Many News:

```
http://localhost:8080/news
```

Get One News:
http://localhost:8080/news/:id

create the .env in the project root with the key:

```
MONGODB_URI="mongodb://{host}:{port}"
```
